# dotfiles

## how to use
run setup.sh to
- add alias to .bashrc
- install pyenv
- install neovim dependencies

```
./setup.sh
```

## alias list
```
ln -s $HOME/.dotfiles/.vim $HOME
ln -s $HOME/.dotfiles/.vimrc $HOME
ln -s $HOME/.dotfiles/.tmux.conf $HOME
ln -s $HOME/.dotfiles/.zshrc $HOME
ln -s $HOME/.dotfiles/.vim/vimrc $HOME/.config/nvim/init.vim
```
